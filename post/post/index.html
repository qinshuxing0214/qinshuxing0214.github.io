How to Use Hugo to Build a Blog

28 Feb 28282, 00:34

Quick Start

Create a Hugo site using the beautiful blackburn theme.

Step 1: Install Hugo

Install Hugo on Windows, and on a machine where the Go compiler tool can run.

There is lots of talk about “Hugo being written in Go”, but you don’t need to install Go to enjoy Hugo. Just grab a precompiled binary!

  1. Download the hugo installation package according to the operating system version.

    Please download from here.

  2. Unzip the compressed package to the specified path, such as C:\hugo_dir in windows system.

  3. Press win+R on the keyboard to bring up the command line terminal.

  4. Enter cmd to open the command window.

  5. Run the following command to enter the hugo installation directory.

    cd C:\hugo_dir

  6. Run the following command to confirm that hugo has been started.

    hugo version

  7. If you see the prompt message of the hugo version number, it means that hugo has been installed successfully.

Step2: Install Git

Git is an open source distributed version control system that can effectively and quickly handle version management from very small to very large projects.

  1. Download the latest version of Git from the official website according to the operating system version. Please download from here.
  2. Double-click the downloaded git installation package.
  3. Install Git according to the prompt.

Step3: Create a New Site

  1. Open the command window.

  2. Run the following command to enter the hugo installation directory.

    cd C:\hugo_dir

  3. Run the following command to create a new site.

    hugo new site myfirstblog

  4. Run the following command to enter the newly created path.

    cd myfirstblog

Step4: Add a Theme

  1. Run the following command to enter the directory where the theme is located.

    cd themes

  2. Run the following command to clone the theme.

    git clone https://github.com/yoshiharuyamashita/blackburn.git

  3. After the theme is cloned, copy the config.toml file in the themes/exampleSite directory to the myfirstblog directory, overwriting the original config.toml file.

  4. Run the following command to run the server.

    hugo server -D

  5. Enter the following address in the browser to verify that the theme is valid.

    http://localhost:1313

    If you can successfully see the running theme, it means that the theme is set successfully.

  6. Press Ctrl+C to stop the server.

Step5：Add some content

Now the blog content is still empty, and you need to add some content.

  1. Run the following command to enter the site directory.

    cd myfirstblog

  2. Run the following command to create the post directory.

    hugo new post/post.md

  3. Use the markdown editing tool to open this newly created post file.

  4. After editing, run the following command to start the server.

    hugo server -D

  5. Enter the local address to view the blog.

    http://localhost:1313/

Step6: Edit and adjust the blog

Your new site already looks great, but you will want to tweak it a little before you release it to the public.

  1. Press Ctrl+C to stop the server.

  2. Open up config.toml in the site directory with a text editor. Edit the baserurl of the blog, title, author, copyright, etc.

    the baserurl needs to be set as a registered github library, such as “qinshuxing0214.github.io”.

    baseurl = "https://www.example.com/"
languageCode = "en-us"
title = "Blackburn Theme Demo"
theme = "blackburn"
author = "Yoshiharu Yamashita"
copyright = "&copy; 2016. All rights reserved."
canonifyurls = true
paginate = 10

Step7: Push to github

Before pushing to github, github repository has been created. For example, qinshuxing0214/qinshxing0214.git.io.

  1. Run the following command to enter the site directory.

    cd myfirstblog

  2. Run the following command to generate static html that can be published.

    hugo --theme=blackburn --baseUrl="https://qinshuxing0214.github.io/" --buildDrafts

  3. Run the following command to enter the generated public folder.

    cd public

  4. Run the following command to create a Git repository.

    git init

  5. Run the following command to add the changes to the cache.

    git add .

  6. Run the following command to submit the modification.

    git commit -m "initial commit"

  7. Run the following command to associate the remote repository.

    git remote add origin https://github.com/qinshuxing0214/qinshuxing0214.github.io.git

  8. Run following command to upload to the remote repository.

    git push -u origin master

  9. In the address bar, enter qinshuxing0214.github.io to visit the blog.
